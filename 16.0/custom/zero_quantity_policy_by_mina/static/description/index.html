Zero or Less Product Quantity Sales and Inventory Transfer Prevention

A module that disallows users to make a sale order or inventory transfer with zero or negative available quantity of products.

An example that shows the available quantity of a given product in a specific inventory location

User may select a product that has no stock in a certain inventory location

A dialogue shows up to inform user that his choice is blocked

But when the user changes the source inventory location which has adequate quantities of this product, no error dialogue is seen

A dialogue shows up to inform user that his choice is blocked if he/she entered a quantity greater than available quantity in the specified source inventory location

In the inventory module, the conditions that are imposed by this module are checked if the operation type is one of types that pulls products from inventory location. This means that opertions that involve receipt only are excluded
